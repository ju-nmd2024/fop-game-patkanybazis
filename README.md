# Guide for the game
## Goal
The player's job is to **help kitty land safely**. This means that kitty's landing **speed should not exceed 12**, otherwise she dies :c

## Controls
### Menu
* **Spacebar:** Start the game
### Game
* **Spacebar:** Jump
* **R:** Restart the game
* **Esc:** Go back to the menu
### Game Over
* **R:** Restart the game
* **Esc:** Go back to the menu
